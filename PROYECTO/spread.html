Spread Operator

Características

En ES5 podíamos encontrar el valor máximo de un array con el método apply en Math.max. 

              

Math.max.apply(null, [-1, 100, 9001, -32]); // 9001

En ES6 podemos usar el operador spread para pasar un arreglo de valores y que sea utilizado en una función como un parámetro. 

              

Math.max(...[-1, 100, 9001, -32]); // 9001

Podemos concatenar arrays fácilmente con esta sintáxis intuitiva. 

              

let cities = ['San Francisco', 'Los Angeles'];
let places = ['Miami', ...cities, 'Chicago']; // ['Miami', 'San Francisco', 'Los Angeles', 'Chicago']
