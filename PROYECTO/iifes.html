IIFEs ahora Blocks

Características

Un uso común de las expresiones de función inmediatamente invocadas (IIFE) es el de envolver variables dentro de su alcance. Ahora con ES6, podemos crear alcances basados en bloque y de esa forma ya no estar limitados a los alcances por función solamente. 

              

(function () {
    var food = 'Meow Mix';
}());

console.log(food); // Reference Error

Usando bloques de ES6. 

              

{
    let food = 'Meow Mix';
};

console.log(food); // Reference Error
