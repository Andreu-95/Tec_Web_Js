Strings

Características

Con ES6, la librería estándar a crecido bastante. En conjunto con estos cambios hay nuevo métodos que pueden ser utilizados con strings como .includes() y .repeat().

includes()
              

var string = 'food';
var substring = 'foo';

console.log(string.indexOf(substring) > -1);

En lugar de esperar un valor que sea mayor a -1 para denotar que se contiene un string podemos usar includes() que devuelve un booleano. 

              

const string = 'food';
const substring = 'foo';

console.log(string.includes(substring)); // true
repeat()
              

function repeat(string, count) {
    var strings = [];
    while(strings.length < count) {
        strings.push(string);
    }
    return strings.join('');
}

Con ES6, tenemos una implementación más sencilla. 

              

// String.repeat(numberOfRepetitions)
'meow'.repeat(3); // 'meowmeowmeow'
